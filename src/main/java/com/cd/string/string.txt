一个字符char占用俩个字节
一个字节byte是8bit，八位带符号的二进制数
所以一个字符char是16位
char 存放范围是0-65535
byte 存放范围是-128-127

byte运算
    1.byte是八位带符号的二进制数。首位0代表正，1代表负
    2.对于负数二进制数转换成其他禁止数，需要对原负二进制数取反+1操作
        对于1000 0001这样的负二进制数取反加一得到结果是0111 1111，0111 1111代表的十进制是127，加上符号就是-127
        即1000 0001就是-127
    3.1000 0000 取模加一 1000 0000=255-127=128
